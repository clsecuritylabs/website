What's a CRP?

A Certified Remediation Pattern is a verified, reproducible mapping from attack to defense.

# Example: CRP-2026-0147 attack: category: prompt_injection reproduction_rate: 0.95 models_affected: [listed under NDA] defense: type: Multi-tier semantic classification verified_block_rate: 0.88 false_positive_rate: 0.02 latency: <15ms avg verification: breach_iterations: 5 block_iterations: 10 judge_panel: 3 vendors, multi-model consensus judge_confidence: 0.94 frameworks: - OWASP LLM01: Prompt Injection - MITRE ATLAS: AML.T0051 - NIST AI RMF: MG-2.1
